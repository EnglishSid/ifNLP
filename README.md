"# ifNLP" 


### Client set-up

1. Download and install Neo4j desktop : https://neo4j.com/download/
2. create new local database (any name, default user/pass)
3. install git
4. install node.js
5. clone this repo


### Package Installation

from the command line : npm install


### Running prototype

node app.js

or if installed nodemon app.js
